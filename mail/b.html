Windows Updateの有効化

[スタート]-[設定]-[更新とセキュリティ]-[更新プログラムのチェック]をクリック
　Microsoft/ Windows Update の利用手順 - Windows 10 の場合
＊OSの再起動を要するメジャー・アップデートは不具合を生じさせることもあるため、更新プログラムの名称をグーグル検索し、安全を確かめてから実行すると良い。

Microsoft Updateの有効化

Office等、Microsoft社の製品全般の更新プログラム。
[スタート]-[設定]-[更新とセキュリティ]-[詳細オプション]-
[Windowsの更新時に他のMicrosoft製品の更新プログラムも入手します」をチェック
